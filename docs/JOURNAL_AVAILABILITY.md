# Journal Availability Text

Use this text near the end of the journal paper, before the bibliography.

```latex
\textbf{Data, code, and artifacts.}
The experiments use public datasets cited in the manuscript.
The source code, experiment configurations, Colab pipeline, final figures, and
LaTeX sources used in this study are available at
\url{https://github.com/Kamran151199/thesis}.
The full paper-artifact workflow writes run logs, generated tables, masking
examples, split/leakage audits, and claim-to-evidence files during execution.
Large checkpoints and downloaded datasets are not stored in the repository.
```

If the review process requires anonymity, replace the public GitHub link with an
anonymous repository link during review and restore the public link after
acceptance.
