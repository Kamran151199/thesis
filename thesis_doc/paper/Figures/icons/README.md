# Thesis Figure Icon Pack

Original SVG line icons for the thesis figures. They are designed for draw.io,
Figma, Inkscape, and LaTeX figure workflows.

Use:

1. Drag an SVG into draw.io, or use `Arrange > Insert > Advanced > SVG`.
2. Keep icons as vectors.
3. Export the final figure as SVG or PDF, not as a screenshot.
4. Recolor in draw.io if needed; the icons use a small neutral palette.

Suggested mapping:

- `image_input.svg`: visual input
- `document_image.svg`: DocVQA/document domain
- `chart_input.svg`: ChartQA/chart domain
- `natural_image.svg`: natural-image VQA domain
- `dataset_stack.svg`: dataset collection or experiment matrix
- `vlm_model.svg`: general VLM block
- `qwen2vl_block.svg`: Qwen2-VL branch
- `blip2_qformer.svg`: BLIP-2/Q-Former branch
- `qlora_adapter.svg`: parameter-efficient adapter
- `contrastive_alignment.svg`: contrastive objective
- `generative_decoding.svg`: generative objective
- `rationale_explanation.svg`: rationale/explanation-aware output
- `masking_grid.svg`: image masking/evidence sensitivity
- `metrics_evaluation.svg`: metrics/evaluation
- `transfer_domains.svg`: domain transfer
- `gpu_resource.svg`: single-GPU/resource constraint
- `checkpoint_refresh.svg`: checkpoint/evaluation refresh
