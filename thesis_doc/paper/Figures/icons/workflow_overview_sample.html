Research controls

Comparison groups keep the model family, data cap, runner, and evaluation path fixed as much as possible.

Failure protection

Intermediate logs and artifacts are saved after each step, so one failed run does not erase completed outputs.

Paper connection

Each result figure and table is tied back to the experiment ledger and claim-to-evidence map before reporting.

Main idea: the notebook is not only a run script; it is an auditable path from data sample to model input, objective, metric, and final paper artifact.
